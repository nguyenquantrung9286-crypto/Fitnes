# 📱 Fitnes — Единая цифровая экосистема здоровья

**Fitnes** — комплексное мобильное приложение, которое объединяет функции персонального умного тренера, диетолога и платформы для связи с профессионалами.

## 🎯 Основная идея
Приложение берет на себя все аспекты здорового образа жизни — от генерации тренировок через ИИ до автоматического анализа питания по фото. Ключевой принцип: **минимум ручного ввода**, максимум автоматизации.

## 🏗️ Технический стек
- **Мобильный фронтенд:** React Native + Expo + TypeScript
- **Стилизация:** NativeWind v4 (TailwindCSS в React Native)
- **Фреймворк навигации:** Expo Router (файл-based routing)
- **Управление состоянием и кэш:** TanStack Query v5 + AsyncStorage (работает офлайн)
- **Backend:** Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **ИИ интеграция:** Polza.ai для распознавания блюд по фото

## 🎮 Ключевой функционал

### 1. Умная регистрация
- 23-параметровый онбординг (рост, вес, ограничения, наличие инвентаря)
- Автоматический расчет индекса массы тела и суточной нормы калорий
- Плавный переход между экранами с индикатором прогресса

### 2. Dashboard (Главная)
- Обзор дневного калоража и макронутриентов
- Интеграция с HealthKit (iOS) для автосбора шагов
- Рекомендации по тренировкам и водному балансу
- Карточки текущих задач и целей

### 3. Тренировки
- Видеоинструкции с техникой выполнения упражнений
- Детальные планы тренировок, сгенерированные ИИ
- Чат с тренером (встроенный мессенджер)
- Возможность ручного управления интервалами и подходами

### 4. AI Food Scanner
- Камера → фото блюда → анализ через нейросеть Polza.ai
- Автоматическое определение объема порции
- Расчет калорийности и раскладки по КБЖУ
- Вывод данных поверх изображения

### 5. Отслеживание прогресса
- Динамические графики изменения веса
- Фотодневник "До и После"
- Визуализация физических показателей с проекцией результатов
- Сравнение прогресса между периодами

### 6. Оффлайн-режим
- Полная работа приложения без интернета
- Локальные напоминания о воде и тренировках
- Автоматическая синхронизация при появлении соединения

## 🎨 Дизайн-система

### Цветовая палитра
- **Основной фиолетовый:** `primary-600: #7C3AED` (оттенки от `primary-50` до `primary-900`)
- **Розовый акцент:** `accent-500: #EC4899`
- **Темная тема:** `dark-700: #18181B` (основной фон), `dark-950: #050509` (глубокий фон)

### Принципы проектирования
- **Крупные кнопки:** Минимум 52px в высоту для удобства одной рукой
- **Текстовая шкала:** `text-sm`, `text-base`, `text-lg`, `text-xl` через TailwindCSS
- **Spacing:** Material Design сетка (8px, 16px, 24px, 32px)
- **Темы:** Светлая/темная/системная с переключением в профиле
- **Принцип нулевого ввода:** Максимальное использование датчиков устройства для автоматического заполнения
- **Органичный дизайн:** Спокойная психологическая среда, скругления, природные формы

### Навигация
Все ключевые разделы закреплены в нижней панели (bottom tab navigation):
- 🏠 Главная (Dashboard)
- 💪 Тренировки (Workouts)
- 📷 Сканер (Scanner)
- 💬 Чат (Chat)
- 👤 Профиль (Profile)

## 🚀 Команды разработки

```bash
# Из директории FitnesApp/

# Development
npx expo start              # Запуск dev сервера
npx expo start -c           # С очисткой кэша
npm run ios                 # Прямо в iOS симулятор
npm run android             # Прямо в Android эмулятор
npm run web                 # Web превью

# В терминале после 'npx expo start':
# i — открыть iOS симулятор
# a — открыть Android эмулятор
# j — React Native Debugger (не Chrome DevTools!)
# w — открыть web

# Database
npx supabase db push --linked --yes              # Запушить миграции
npx supabase db dump --linked --schema public    # Дамп схемы

# Production
eas build -p ios --profile preview
eas build -p android --profile preview
```
